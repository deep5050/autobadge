# autobadge
Simple CLI tool to Generate Essential Repository Badges with Ease.
It Asks for minimum informtion from the user and produces all (almost) the essential badges.

## Supports
- [x] MarkDown
- [x] HTML
- [x] Shields Badges
- [ ] Badgen Badges
- [ ] ForTheBadge Badges

## Style
- [x] Flat
- [x] Flat-Square
- [x] Plastic
- [x] Social
- [x] For-The-Badge
